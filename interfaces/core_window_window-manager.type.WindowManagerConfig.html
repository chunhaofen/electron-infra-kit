    类型声明

    • 可选autoInit?: boolean

      Whether to automatically initialize IPC on instantiation, default is true 是否在实例化时自动初始化 IPC，默认为 true

    • 可选channel?: string

      Async communication channel name, default is 'renderer-to-main' 异步通信频道名称，默认为 'renderer-to-main'

    • 可选syncChannel?: string

      Sync communication channel name, default is 'renderer-to-main-sync' 同步通信频道名称，默认为 'renderer-to-main-sync'
